## git 常用命令（VS code）
| 步骤        | 命令                     | 说明         |
| --------- | ---------------------- | ---------- |
| 克隆项目      | `git clone <url>`      | 从GitHub下载  |
| 进入目录      | `cd 项目名`               | 切换到项目文件夹   |
| 打开编辑器     | `code .`               | 用VS Code打开 |
| 查看更改      | `git status`           | 看哪些文件修改了   |
| 添加更改      | `git add .`            | 把所有修改加入暂存  |
| 提交更改      | `git commit -m "说明"`   | 保存修改记录     |
| 推送到GitHub | `git push origin main` | 上传修改       |
